Chinese notes

  1. 什么是 TypeScript

    TS 是 JavaScript 的超集（superset）。TS 采用 JS 并为其添加了新的特性和语法。TS 不能在 JS 环境（如浏览器，node.js）直接执行。

    TS 是一种编程语言，也是一种工具。它是一个强大的编译器，它会把 TS 代码编译成 JS 代码。也就是说，程序员编写了具有所有新特性和所有优点的 TS 代码，结果得到的是正常的 JS 代码。

    TS 给了程序员一个机会，其可以在脚本运行之前，在浏览器运行时发现代码中的错误。也就是说，TS 在项目开发过程中 (compliation) 提供了额外的错误检查。这可以在早期捕获并修复错误。JS 只能在 runtime 时发现错误。

  2. TypeScript overview

    1. Types：可以避免许多错误。

    2. Next-gen JS features：使用现代 JS 特性，并仍然可以生成和发布在旧浏览器中工作的代码。

    3. TS 专属特性：如 interfaces 和 Generics（泛型）。

    4. Meta-Programming 特性：如 Decorators。

    5. Rich Configuration Option：TS 可以配置，根据要求微调，让它更严格或者宽松。

    6. 可以在非 TS 项目中获得支持：需要通过现代工具和现代 IDE。

  4. 一元加操作符（+）

    在 JavaScript 中，将一个值转换为数字可以使用一元加操作符（+）。当在一个非数字值之前使用一元加操作符时，JavaScript 会尝试将该值转换为一个数字。如果值不能转换为数字，则会返回 NaN。

    因此，如果在 JavaScript 中使用 +input.value，它会将 input 元素的值（通常是一个字符串）尝试转换为数字类型。这可以用于确保从 input 元素中获取到的值是数字类型，而不是字符串类型，以便在后续计算中进行正确的数值计算。
