Types - union, literal, custom/aliases

2021-01-05
2023-03-01
Chinese notes

  1. union 联合类型

    写法为 参数: 类型1|类型2|类型3...。表示参数可以赋这些类型的值。如：

    例子中，无法在函数内直接写 const result = input1 + input2，这是因为 TS 不会分析联合类型中有什么，因此无法检测这些值是否能进行 + 运算。因此要自己在函数内写 if(typeof ...) 类型检测代码。

    union 类型的使用场景通常是，让函数可以更灵活地接收参数，但随后会根据所获取的确切类型在函数中使用不同的逻辑。这样函数就可以处理多种不同类型的值了。但这也不是必须的。也会出现使用 union 但不做后续类型判断的场景，这取决于函数逻辑。

    union 可以用来联合 type, class 和 interface。因为它们在 TS 中都是 type。联合后可以用 type a = 取值。

  2. Literal 字面量类型
